# Installation

Add this to `~/.config/alacritty/alacritty.yml`

## Light

```yaml
colors:
  primary:
    background: "0xfafafa"
    foreground: "0x5e646f"

  normal:
    black: "0x5e646f"
    red: "0xdb7070"
    green: "0x7c9f4b"
    yellow: "0xd69822"
    blue: "0x6587bf"
    magenta: "0xb870ce"
    cyan: "0x509c93"
    white: "0x949ba5"

  bright:
    black: "0x5e646f"
    red: "0xdb7070"
    green: "0x7c9f4b"
    yellow: "0xd69822"
    blue: "0x6587bf"
    magenta: "0xb870ce"
    cyan: "0x509c93"
    white: "0x949ba5"
```
